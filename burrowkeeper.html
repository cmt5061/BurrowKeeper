📖 How to set up Firebase (free) ▼
1
Create a Firebase project
Go to console.firebase.google.com → Click "Add project" → Enter a name → Continue through the setup wizard.
2
Create a Realtime Database
In your project, go to Build → Realtime Database → Click "Create Database" → Choose a region → Start in test mode.
3
Apply the security rules
In the Realtime Database console, click the "Rules" tab → Replace the contents with the rules shown in Settings → Click "Publish".
4
Copy your database URL
In the Realtime Database "Data" tab, copy the URL at the top — it looks like https://your-project-rtdb.firebaseio.com — paste it in the field above and click Save.
5
Sync across devices
Share your Sync Key with family members. They enter it on any device to access and update your shared home data.

📖 How to set up Firebase (free) ▼
1
Create a Firebase project
Go to console.firebase.google.com → Click "Add project" → Enter a name (e.g. "BurrowKeeper") → Continue through the setup wizard.
2
Create a Realtime Database
In your project, go to Build → Realtime Database → Click "Create Database" → Choose a region → Start in test mode (we'll add proper rules next).
3
Apply the security rules
In the Realtime Database console, click the "Rules" tab → Replace the contents with the rules shown above → Click "Publish".
4
Copy your database URL
In the Realtime Database "Data" tab, copy the URL at the top — it looks like https://your-project-rtdb.firebaseio.com — paste it in the field above and click Save.
5
Sync across devices
Share your Household Sync Key with family members. They enter it on any device to access and update your shared home data.
